# members_intro

이 폴더는 **토요일 첫 모임에서** 각자 자기소개 마크다운 파일을 PR로 올리는 실습 공간입니다.

This folder is where each member submits their intro markdown via PR **during the first Saturday meetup**.

## 규칙 / Rules

- 파일명: `이름.md` (예: `byeol.md`, `jina.md`, `sein.md`, `inchan.md`)
- 템플릿: [`TEMPLATE.md`](TEMPLATE.md) 참고
- 브랜치명: `intro/이름`
- PR 제목: `자기소개: 이름 / Intro: name`

## 진행 절차 / Steps

토요일 실습 절차는 [`../docs/saturday-mission.md`](../docs/saturday-mission.md) 를 따라주세요.
Follow [`../docs/saturday-mission.md`](../docs/saturday-mission.md) for the full Saturday walkthrough.
